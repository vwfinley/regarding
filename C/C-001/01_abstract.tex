Setting up a modern developer environment is a non-obvious undertaking. 
General instructions regarding how to setup a working software development environment are discussed herein.

Instructions pertain to an environment where:
\begin{itemize}
  \item A local IDE is connected to a remote VM, and 
  \item The VM is configured to run a developer container.
\end{itemize}

Instructions are specific to the author's containerized environment.
However, the instructions are can be adapted to other project environments.
