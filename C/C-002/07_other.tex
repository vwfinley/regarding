\section{Other Setup}
For the Ubuntu Server distro, the remaining steps to set up a development enviroment are just like for the Ubuntu Desktop distro.

To complete setup, follow these sections from the previous article:
\itemize{begin}
  \item Section 7.2: Use the commandline to update software on the VM.
  \item Section 8: Save the VM's name and static IP address (from previous section) in the Hosts file. 
  \item Section 10 thru 17: Finish setting up the development environment.
\itemize{end}
