\section{Other Setup}
For the Ubuntu Server distro, the remaining steps to set up a development environment are same as for the Ubuntu Desktop distro.

To complete the setup, follow sections from the previous article:
\begin{itemize}
  \item Section 7.2: Use the commandline to update software on the VM.
  \item Section 8: Save the VM's name and static IP address in the hosts file. 
  \item Section 10 thru 17: Finish setting up the development environment.
\end{itemize}
